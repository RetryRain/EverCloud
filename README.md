# EverCloud ☁️

EverCloud is a full-stack cloud hosting platform designed to simulate real-world hosting infrastructure, including billing, support, and system management workflows.

## 🚀 Features

- 🧾 Billing System – Manage payments and subscriptions
- 💬 Integrated Support – Built-in communication system for user support
- ⚙️ Admin Controls – Manage users and hosting services
- 🧱 Scalable Architecture – Layered architecture with clear separation of concerns
- 🔐 Secure Backend – Structured service and domain layers for maintainability

## 🏗️ Architecture

The application follows a clean layered architecture:

- **Domain Layer** – Core business models and logic  
- **Service Layer** – Business rules and operations  
- **Data Access Layer** – Database interactions  
- **Presentation Layer** – ASP.NET frontend  

## 🛠️ Tech Stack

- ASP.NET  
- C#  
- jQuery  
- SQL Server  

## 📌 Why This Project?

This project was built to replicate a real-world cloud hosting system, focusing on:

- Backend architecture design  
- Full-stack system integration  
- Scalability and maintainability
